Mode: interactive

Syntax

atomsk

Description

Running the program without any argument will enter interactive mode, which offers a command-line interpreter.

In this mode the user can enter commands, and they are executed in real-time. The available commands are:

  • help Displays a summary of commands
  • ls Displays a list of files in the current directory
  • read <file> Read the <file> and load its content in memory
  • memory Displays a summary of the memory used
  • quit Exit this mode and close atomsk
  • write <file> Write the current system into <file>
  • memory Displays a summary of the memory used

The options of atomsk are also available. They must be called without the leading minus sign (-), and with all their parameters.

Note that only one command (or option) can be issued at a time.

When running in interactive mode the verbosity level is automatically reset to 1 if it is set to 0 or 2.
