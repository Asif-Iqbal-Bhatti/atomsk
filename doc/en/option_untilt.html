Option: untilt

Syntax

-untilt [<xy|xz|yz|yx|zx|zy>]

Description

This option deforms the system (box and atoms) to remove any tilt of the box. This can be useful when a system was sheared, and one wants to un-shear it.

After this option is applied, non-diagonal components of the box vector matrix become zero, so that the box vectors become H1=(h1,0,0), H2=(0,h2,0), H3=(0,0,h3). In other words the box becomes orthogonal.

It is possible to remove only one tilt component by specifying 'xy', 'xz', 'yz', 'zx', or 'zy'.

Default

By default the system is not untilted.

Examples

  • atomsk initial.cfg -untilt final.xyz

    This will read the system initial.cfg, and remove any tilt. The result will be output into final.xyz.

  • atomsk initial.cfg -untilt xy final.xyz

    This will read the system initial.cfg, and remove the xy tilt, i.e. the component H(2,1) will become zero. The result will be output into final.xyz.
