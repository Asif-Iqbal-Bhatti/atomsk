Option: swap

Syntax

-swap <id1> <id2>

Description

This will swap the positions of the two atoms of the given indices.

If shells (in the sense of an ionic core-shell model) and/or auxiliary properties are defined, then they will also be swapped.

Default

By default no atom is swapped.

Examples

  • atomsk initial.cfg -swap 530 1065 final.xsf

    This will read the file initial.cfg, and exchange the positions and all properties of atoms 530 and 1065. The final result will be written into final.xsf.
