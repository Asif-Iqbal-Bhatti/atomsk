Format: CEL

Name: Dr Probe Cell File

Extension: .cel

Specification: Website of Dr Probe

Visualization programs: Dr Probe

Restrictions

Dr Probe is a software for simulating high-resolution scanning or transmission electron microscopy (HRTEM) images. Atomsk can read and write CEL files, with the restrictions below.

To write this file format, the occupancy and thermal vibration should be defined as auxiliary properties for each atom. If they are not set, Atomsk will use default values of 1 and 0, respectively.

It should be noted that Atomsk does not perform any implicit conversion of units. By default CEL files use nanometers as unit of length (and nm2 for thermal vibration), and Atomsk stores these values without modifying them. When reading a CEL file, one can convert the values to another unit with the option -unit. When writing a CEL file, one should convert the units to nm.

Examples

  • atomsk my_system.xsf -unit A nm cel

    This will read the file my_system.xsf, convert the units to and convert it to my_system.cel.
