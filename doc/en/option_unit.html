Option: unit

Syntax

-unit <unit1> <unit2>

-u <unit1> <unit2>

Description

This converts atom coordinates or velocities from one unit system <unit1> to another <unit2>.

The supported units of distance are:

  • km
  • m
  • mm
  • micron or µm
  • nm
  • A or Angstroms or angstroms
  • B or Bohr or bohr
  • pm
  • fm

The supported units of time are:

  • h
  • min
  • s
  • ms
  • µs
  • ns
  • ps
  • fs

Units of velocity must be specified as the concatenation of a unit of distance, a slash symbol (/), and a unit of time (see examples below).

Note that atomsk will perform the conversion blindly, as it has no way to know in which unit the coordinates are written in the input file. It is up to the user to make sure that the <unit1> actually corresponds to what is written in the input file.

Default

By default the units are not changed.

Examples

  • atomsk initial.xyz -unit Bohr Angstroms final.xsf

    This will read the file initial.xyz, where atom coordinates are written in units of Bohr radius, convert them to angstroms, and output the result into final.xsf.

  • atomsk initial.xsf final.xyz -u A nm

    This will convert from Angstroms to nanometers.

  • atomsk initial.lmc final.cfg -u A/ps nm/ps

    This will convert atom velocities from Angstroms per picosecond (Å/ps) to nanometers per picosecond (nm/ps).

  • atomsk initial.lmc final.cfg -u nm/fs km/h

    This will convert atom velocities from nanometers per femtosecond (nm/fs) into kilometers per hour (km/h).
