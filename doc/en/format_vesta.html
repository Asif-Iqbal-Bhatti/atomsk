Format: VESTA

Name: VESTA format

Extension: vesta

Specification: VESTA documentation

Visualization programs: VESTA

Restrictions

Atomsk can read VESTA files. If the file contains symmetry operations (after the keyword "SYMOP"), then they are applied immediately to all atoms.

When writing a VESTA file, Atomsk ignores symmetry operations, i.e. the space group will be P1. If forces exist as auxiliary properties (fx, fy, fz) then they are written as vectors after the position of each atom. Alternatively if atom velocities exist, they are written as vectors. Otherwise the vectors will be zero for all atoms.

Examples

  • atomsk initial.cfg final.vesta

    This will read the file initial.cfg, and convert it to the file final.vesta.

  • atomsk CONTCAR vesta

    This will read the VASP file CONTCAR, and convert it to the file CONTCAR.vesta.

  • atomsk file.vesta lammps

    This will read the VESTA file file.vesta, and convert it to the file file.lmp.
