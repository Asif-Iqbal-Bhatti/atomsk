Program behaviour

By default the program atomsk tries to be polite: it asks if files should be overwritten, but it does not overflow the user with useless information. You can change the default behaviour with the command-line parameters below, or by setting up a configuration file in your home directory.

These parameters will apply all along a run if they appear anywhere in the command-line.

  • ignore - Always ignore existing files
  • language - Set language of messages
  • logfile - Set up log file name
  • overwrite - Always overwrite existing files
  • verbosity - Set verbosity level
