Format : VESTA

Nom : fichier VESTA

Extension : vesta

Spécification : documentation de VESTA

Programmes de visualisation : VESTA

Restrictions

Atomsk peut lire les fichiers VESTA. Si le fichier contient des opérations de symétrie (après le mot-clé "SYMOP"), alors elles sont appliquées immédiatement.

Lorsqu'il écrit un fichier VESTA, Atomsk ignore aussi les opérations de symétrie, c'est-à-dire que le groupe d'espace sera P1. Si les forces sont définies en tant que propriétés auxiliaires (fx, fy, fz), alors elles sont écrites comme vecteur après la position de chaque atome. Sinon, si les vitesses des atomes sont définies alors elles sont utilisées comme vecteurs. Sinon les vecteurs seront nuls pour tous les atomes.

Exemples

  • atomsk initial.cfg final.vesta

    Ceci va lire le fichier initial.cfg, et le convertira en final.vesta.

  • atomsk CONTCAR vesta

    Ceci va lire le fichier VASP CONTCAR, et le convertira en CONTCAR.vesta.

  • atomsk file.vesta lammps

    Ceci va lire le fichier VESTA file.vesta, et le convertira en file.lmp.
