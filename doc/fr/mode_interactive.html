Mode : interactif

Syntaxe

atomsk

Description

Si le programme est exécuté sans aucun paramètre en ligne de commandes, il offrira un interpréteur de commandes.

Dans ce mode l'utilisateur peut entrer des commandes, lesquelles sont interprétées et exécutées en temps réel. Les commandes disponibles sont :

  • help Affiche une liste des commandes disponibles
  • ls Affiche une liste des fichiers du dossier courant
  • read <fichier> Lit le <fichier> et le charge en mémoire
  • memory Affiche un résumé de l'utilisation mémoire courante
  • quit Sort de ce mode et ferme atomsk
  • write <fichier> Ecrit le système courant dans le <fichier>

Les options d'atomsk sont également disponibles. Elles doivent être appelées sans le signe moins (-) et avec tous leurs paramètres.

Notez que les instructions (ou options) doivent être entrées les unes après les autres.

Lorsque le programme est exécuté en mode interactif le niveau de verbosité est automatiquement égal à 1.
