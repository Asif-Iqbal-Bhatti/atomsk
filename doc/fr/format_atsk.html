Format: ATSK

Nom : fichier binaire d'atomsk

Extension : .atsk

Spécification : voir le code source d'atomsk

Programmes de visualisation : aucun

Restrictions

Ce format de fichier est spécifique à atomsk. Il enregistre les vecteurs de boîte, les positions atomiques, les positions des coquiles (dans le cadre d'un potentiel ionique cœur-coquille), et les propriétés auxiliaires des atomes.

Puisque c'est un format binaire, il utilise généralement moins d'espace disque que les fichiers ASCII (ou fichiers texte), il est donc plus rapide de lire et écrire des fichiers ATSK. Cependant cela signifie aussi qu'il est impossible de lire ce type de fichiers dans un éditeur de texte. Ce format peut être utile pour lire/écrire des fichiers intermédiaires, par exemple lorsqu'il est nécessaire d'invoquer atomsk plusieurs fois de suite pour construire un système atomique, surtout si le système est grand (plusieurs centaines de milliers d'atomes).

NOTE IMPORTANTE : Ce format de fichiers est conçu pour être utilisé uniquement par atomsk. Il est très improbable que ce format soit un jour lisible par d'autres programmes, il peut même être impossible de lire un fichier ATSK sur une machine différente de celle où il a été produit, ou avec une version différente d'atomsk. L'utilisation de ce format devrait se restreindre aux fichiers temporaires ou intermédiaires. Il est fortement déconseillé de sauvegarder ou d'archiver des fichiers dans ce format.

Exemples

  • atomsk mon_systeme.cfg atsk

    Ceci va lire le fichier mon_systeme.cfg, et le convertira en mon_systeme.atsk.
