Option: remove random atoms

Syntax

-remove-random-atoms <N> <species>

-rmrand <N> <species>

Description

This option randomly removes N atoms of the given <species> from the system.

If <species> is set to "any" or "all" then N atoms will be randomly removed from the system regardless of their species.

Since this option attempts to generate random indices of the atoms to be removed it is largely non-reproducible, i.e. starting from the exact same system and applying the present option will likely lead to different results.

If atoms are selected with the option -region, then only atoms from that region will be removed at random.

Note that if the removed atoms had shells (in the sense of a core-shell model) and/or auxiliary properties, they are also deleted. In addition, some of the remaining atoms may have a different index after this option is applied.

Default

By default no atom is removed.

Examples

  • atomsk initial.cfg -rmrand 12 Fe final.xsf

    This will read the file initial.cfg, remove 12 iron atoms randomly from the system, and output the result to final.xsf.

  • atomsk initial.cfg -rmrand 3 any final.xyz

    This will randomly remove 3 atoms from the system.
