Option : unskew

Syntaxe

-unskew

Description

Cette option permet de "détordre" la boîte. Soient les vecteurs de boîte suivants :

| H(1,1) H(1,2) H(1,3) |
| H(2,1) H(2,2) H(2,3) |
| H(3,1) H(3,2) H(3,3) |

Si un élément non-diagonal a une valeur supérieure à son composant principal (c-à-d. si |H(i,j)|>0.5*H(j,j) pour i≠j) alors la boîte est considérée comme "tordue". Cette option "détord" la boîte (c-à-d. ajoute ou soustrait H(j,j) à la composante inciminée).

Notez que les coordonnées (cartésiennes) des atomes ne sont pas modifiées par cette option, certains atomes peuvent donc se retrouver à l'extérieur de la nouvelle boîte. Pour les replacer à l'intérieur de la boîte il est possible d'utiliser l'option -wrap.

Défaut

Par défaut la boîte n'est pas détordue.

Exemples

  • atomsk system.xsf -unskew lmp

    Ceci lira le fichier system.xsf, détordra la boîte, et écrira le résultat final dans le fichier de données de LAMMPS system.lmp.
