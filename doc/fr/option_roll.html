Option: roll

Syntax

-roll <axe> <θ>

Description

Cette option permet d'« enrouler » le système, c'est-à-dire le plier pour former un arc de cercle, autour d'un axe cartésien.

The user has to provide the axis of rotation (has to be x, y or z -referring to cartesian axes), and the angle of rotation θ (in degrees). An angle θ=0° means no bending, while θ=360° means rolling the system to form a tube.

For large angles it is recommended that the system is pseudo-2D, i.e. very thin in one direction normal to the <axis>.

If forces (fx,fy,fz) or velocities (vx,vy,vz) exist as auxiliary properties, then they will be rotated accordingly.

If a selection was defined (with the option -select) then the rotation will be applied only to selected atoms, and the box will not be rotated. This can result in some atoms being out of the box, which can be fixed thanks to the option -wrap.

This option does not change the dimensions of the cell, nor the elastic tensor if it is defined.

Défaut

Par défaut le système n'est pas enroulé.

Exemples

  • atomsk initial.cfg -roll Z 10 final.xyz

    This will bend the system to form an angle of 10 degrees around Z.

  • atomsk initial.cfg -roll Y 360 final.xyz

    This will roll the system so that it forms a closed cylinder along the Y axis.
