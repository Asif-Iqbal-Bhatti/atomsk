Option : swap

Syntaxe

-swap <id1> <id2>

Description

Cette options échange les positions des atomes ayant les deux indices donnés.

Si des coquilles (dans le cadre d'un modèle ionique cœur-coquille) et/ou des propriétés auxiliaires sont définies, alors elles seront aussi échangées.

Défaut

Par défaut aucun atome n'est échangé.

Exemples

  • atomsk initial.cfg -swap 530 1065 final.xsf

    Ceci va lire le fichier initial.cfg, et échangera les atomes 530 et 1065. Le résultat final sera écrit dans le fichier final.xsf.
