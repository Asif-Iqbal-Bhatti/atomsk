Option : rotate

Syntaxe

-rotate <axe> <θ>

-rot <axe> <θ>

Description

Cette option permet de pivoter le système autour d'un axe cartésien.

Il est nécessaire de fournir l'axe de rotation (x, y ou z), et l'angle de rotation θ (en degrés).

Le centre de rotation est l'origine (0,0,0). Pour effectuer une rotation autour d'un autre point il est possible d'employer l'option -shift pour amener ce point en (0,0,0), puis appeler la présente option.

Si les forces (fx,fy,fz) ou les vitesses (vx,vy,vz) existent comme propriétés auxiliaires, alors la rotation s'appliquera aussi à ces vecteurs.

Cette option affectera certaines propriétés du système (par ex. le tenseur élastique) si elles avaient été précédemment définies par l'option -properties.

Si une sélection est définie (avec l'option -select) alors la rotation ne sera appliquée qu'aux atomes sélectionnés.

Défaut

Par défaut le système n'est aps pivoté.

Exemples

  • atomsk initial.cfg -rotate x 45 final.xyz

    Ceci tournera le système de 45 degrés autour de l'axe X.
