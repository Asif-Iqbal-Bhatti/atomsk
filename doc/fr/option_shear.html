Option : shear

Syntaxe

-shear <normale> <amplitude> <direction>

Description

Cette option applique un cisaillement simple au système, en inclinant un axe suivant une direction donnée. Les paramètres associés à cette option sont :

  • normale : l'axe qui sera incliné (doit être "x", "y" ou "z") ; le vecteur d'inclinaison sera contenu dans le plan normal à cet axe.
  • amplitude : l'amplitude du vecteur d'inclinaison (en Å), ou valeur de déformation (en %) ; elle peut être positive ou négative.
  • direction : direction du vecteur d'inclinaison ("x", "y" ou "z").

Par exemple, si <normale>=Y, <amplitude>=0.1 Å et <direction>=X, alors le second vecteur de boîte [H(2,1) H(2,2) H(2,3)] sera incliné pour devenir [H(2,1)+0.1 H(2,2) H(2,3)]. Les coordonnées des atomes sont aussi ajustées en conséquence.

Si la hauteur de boîte suivant la <normale> est <h>, l'angle θ duquel la <normale> est inclinée est définie par : tan θ = <shear amp>/<h>.

Notez que si <normale> = <direction> alors une déformation normale sera appliquée, et non un cisaillement. Pour une déformation normale il est préférable d'employer l'option -deform.

Défaut

Par défaut aucun cisaillement n'est appliqué au système.

Exemples

  • atomsk original.xsf -shear x 1.5 z final.xyz

    Ceci inclinera le premier vecteur de boîte de 1.5 Å suivant Z.

  • atomsk original.xsf -shear x 0.5% z final.xyz

    Ceci appliquera un cisaillement de εxz=0.5% au système.
